= Redmine Cancel Button

Finally you can stop the update notes

http://git.zettabajt.pl:redmine_plugins/cancel_button.git

=== Plugin installation

1  Change the name directory to: redmine_cancel_button

2. Copy the plugin directory into the:
	(Redmine 1.x) vendor/plugins
	(Redmine 2.x) plugins

3. Migrate plugin:
   (Redmine 1.x) rake db:migrate_plugins RAILS_ENV=production
   (Redmine 2.x) rake redmine:plugins:migrate RAILS_ENV=production

4. Start Redmine
