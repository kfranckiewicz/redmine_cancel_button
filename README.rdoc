= Redmine Cancel Button

Finally you can cancel the process update notes.
Plugin was tested on version 2.3.4
Supported languages(locales): from redmine core

=== Plugin installation

Download the plugin from: 

    git clone http://git.zettabajt.pl/redmine_plugins/redmine_cancel_button.git
    git clone https://github.com/kfranckiewicz/redmine_cancel_button.git

1. Copy the plugin directory into the:

    plugins

2. Start/Restart Redmine
